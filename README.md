A chunk-based analytical query engine in pure Python and NumPy.
Processes 1M+ row datasets within limited RAM where Pandas crashes.

Built as a Year 1 university project.

## Stack
- Python 3.10+
- NumPy only (no Pandas, no external dependencies)
